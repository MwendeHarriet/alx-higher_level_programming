#1-my_list.txt
This module inherits from a list.

Usage
=====
    >>> MyList = __import__('1-my_list').MyList


Create New Instance
==================

    >>> my_list = MyList()

Empty list
=========

    >>> print(my_list)
    []

    >>> my_list.print_sorted()
    []

Arguments
========

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

New Instance
===========

    >>> print(my_list)
    [1, 4, 2, 3, 5]

 `print_sorted()` method
========================

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Negative Numbers
===============

    >>> my_list.append(-4)
    >>> my_list.append(-9)

    >>> print(my_list)
    [1, -2, 3, -4 ,5 -6]

    >>> my_list.print_sorted()
    [-6, -4, -2, 1, 3, 5]
